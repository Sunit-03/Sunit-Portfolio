## 🧩 Portfolio Architecture

This portfolio itself represents my aesthetic and architectural standards:
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS & Glassmorphism
- **Animations:** Framer Motion
- **Components:** shadcn/ui & Lucide Icons
- **Features:** Dark/Light theme, Smooth scrolling, Mobile-first responsive design

### 🛠️ Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sunit-03/Sunit-Portfolio.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run development server:**
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000)
